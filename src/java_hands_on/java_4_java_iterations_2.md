# Problem : Factorial of Input Numbers

Write a program to read 5 numbers and print factorial of each.
(Final answer should be non-decimal numbers.)

Example: <br/>
<i><b>Input: </b></i>
<br/>
2 <br/>
3 <br/>
4 <br/>
6 <br/>
5

<i><b>Output: </b></i>
<br/>
2 <br/>
6 <br/>
24 <br/>
720 <br/>
120

Important: Answer is not 'e' since 'H' has lower ASCII value than 'e'.
